# KebabMenu

케밥 메뉴를 나타내는 공용 컴포넌트입니다. 기본 메뉴 박스를 제공하고, `MenuListBox` props로 전달할 수 있습니다.
해당 props로 커스텀 메뉴 박스 컴포넌트를 전달할 수 있습니다.
